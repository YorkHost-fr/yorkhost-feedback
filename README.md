## 🇬🇧 English

### Welcome to YorkHost Feedback

This repository is the official place to report bugs, request features, and provide feedback for **YorkHost** services.

YorkHost is a French cloud hosting company specializing in:
- 🎮 Game Servers (FiveM, Minecraft, Palworld, Valheim, and more)
- 💻 VPS Hosting
- 🌐 Web Hosting

### How to Submit Feedback

#### 🐛 Report a Bug

1. Check if the bug has already been reported in [existing issues](../../issues)
2. If not, [create a new issue](../../issues/new) using the **Bug Report** template
3. Provide as much detail as possible:
   - Service affected (Game Server, VPS, Web Hosting)
   - Your server ID or service reference
   - Steps to reproduce the issue
   - Expected vs actual behavior
   - Screenshots if applicable

#### 💡 Request a Feature

1. Check if the feature has already been suggested in [existing issues](../../issues)
2. If not, [create a new issue](../../issues/new) using the **Feature Request** template
3. Describe your idea clearly and explain why it would be useful

#### 💬 General Feedback

For general questions or discussions, feel free to open an issue with the **Feedback** label.

### Guidelines

- ✅ Be respectful and constructive
- ✅ One issue per report
- ✅ Search before posting to avoid duplicates
- ❌ Do not share sensitive information (passwords, personal data)
- ❌ Do not share server IPs or confidential configurations

### Response Time

Our team reviews feedback regularly. While we can't guarantee implementation of every suggestion, we carefully consider all feedback to improve our services.

### Contact

- 🌐 Website: [yorkhost.fr](https://yorkhost.fr)
- 📧 Support: support@yorkhost.fr
- 💬 Discord: [Join our community](https://discord.gg/yorkhost)

## 🇫🇷 Français

### Bienvenue sur YorkHost Feedback

Ce dépôt est l'espace officiel pour signaler des bugs, demander des fonctionnalités et partager vos retours sur les services **YorkHost**.

YorkHost est une entreprise française d'hébergement cloud spécialisée dans :
- 🎮 Serveurs de jeux (FiveM, Minecraft, Palworld, Valheim, et plus)
- 💻 Hébergement VPS
- 🌐 Hébergement Web

### Comment soumettre un retour

#### 🐛 Signaler un bug

1. Vérifiez si le bug a déjà été signalé dans les [issues existantes](../../issues)
2. Si non, [créez une nouvelle issue](../../issues/new) en utilisant le modèle **Bug Report**
3. Fournissez un maximum de détails :
   - Service concerné (Serveur de jeu, VPS, Hébergement Web)
   - Votre ID serveur ou référence de service
   - Étapes pour reproduire le problème
   - Comportement attendu vs comportement actuel
   - Captures d'écran si possible

#### 💡 Demander une fonctionnalité

1. Vérifiez si la fonctionnalité a déjà été suggérée dans les [issues existantes](../../issues)
2. Si non, [créez une nouvelle issue](../../issues/new) en utilisant le modèle **Feature Request**
3. Décrivez clairement votre idée et expliquez pourquoi elle serait utile

#### 💬 Retour général

Pour les questions générales ou discussions, n'hésitez pas à ouvrir une issue avec le label **Feedback**.

### Règles de bonne conduite

- ✅ Soyez respectueux et constructif
- ✅ Une issue par signalement
- ✅ Recherchez avant de poster pour éviter les doublons
- ❌ Ne partagez pas d'informations sensibles (mots de passe, données personnelles)
- ❌ Ne partagez pas les IPs de serveurs ou configurations confidentielles

### Délai de réponse

Notre équipe examine régulièrement les retours. Bien que nous ne puissions pas garantir l'implémentation de chaque suggestion, nous considérons attentivement tous les retours pour améliorer nos services.
